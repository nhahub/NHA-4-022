# Plant Health Detector

## 🎯 Objective
To develop an automated diagnostic system using **Deep Learning** and **Computer Vision** to classify plant leaves as **Healthy** or **Diseased**.
The goal is to provide a scalable tool for early stress detection (pests, fungi, or nutrient deficiency) to reduce crop loss.

## 🛠️ Technical Stack
* **Language:** Python.
* **DL Framework:** TensorFlow / Keras or PyTorch.
* **Computer Vision:** OpenCV (cv2), PIL.
* **Data Science:** NumPy, Pandas, Matplotlib.
* **Deployment:** Flask/FastAPI or Streamlit.

## ⚙️ System Pipeline
1.  **Data Ingestion:** Image acquisition and cleaning.
2.  **Preprocessing:** Resizing ($224\times224$), normalization, and augmentation.
3.  **Core Engine:** Convolutional Neural Networks (CNN) for feature extraction and classification.
4.  **Interpretability:** Utilizing class activation maps to identify specific disease markers on the leaf.

## 📅 Roadmap & Deliverables
* **Phase 1:** Dataset collection (PlantVillage), EDA, and preprocessing.
* **Phase 2:** Advanced color channel analysis and CNN architecture selection.
* **Phase 3:** Model training, optimization, and evaluation (Accuracy, F1-score).
* **Phase 4:** Deployment via web API and MLOps monitoring setup.
